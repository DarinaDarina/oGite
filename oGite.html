Git-Система контроля версий

Есть 3 типа систем контроля версий

  • локальная
  • Централизованная
  • децентрализованная

первоначальная настройка

git config --global user.name "VASYA"

git config --global user.email "vasya@mail.com"

git init - Создание нового репозитория

git status - покажет какие файлы есть, и что их необходимо добавить в репозиторий

git add . - добавит все файлы

git add имя файла - добавит файл с указаным именем

git commit -m "кракий комментарий" - сделает Снимок нашео проекта на данный момент

git push origin master - закинет файл в репозиторий
